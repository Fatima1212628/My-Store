# E-Commerce Store

A full-stack Django e-commerce application with product listings, a shopping cart, user authentication, and order processing.

## Features

- ✅ Product Listing with Images (via Picsum/Unsplash placeholders)
- ✅ Product Details Page
- ✅ Shopping Cart (Session-based)
- ✅ User Registration & Login (Customized auth templates)
- ✅ Checkout Process with Order Management
- ✅ Order History for logged-in users
- ✅ Responsive Design

## Technologies Used

- **Backend:** Django 6.0, Python 3.13
- **Database:** SQLite
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript
